# Enviro 365 Investments Project
This project is a full-stack investment management system developed as part of the Enviro365 Investments Junior Developer Technical Assessment.

The application automates the withdrawal notice process by enabling investors to view their investment portfolios, submit withdrawal requests, and download withdrawal statements as CSV reports. The solution follows modern enterprise development practices with a Spring Boot REST API backend and an Angular frontend.

The system enforces business validation rules to ensure withdrawals are processed correctly while providing a clean, user-friendly interface and reliable backend services.

## Objectives

- Provide investors with access to their investment portfolios.
- Allow submission of withdrawal notices with automatic balance validation.
- Generate downloadable CSV withdrawal statements with filtering capabilities.
- Demonstrate clean architecture, validation, testing, monitoring, and maintainable code.
